# Tic-Tac-Toe Game 🎮

## 📌 Project Description
This is a simple Tic-Tac-Toe game built using basic programming concepts. The game is played on a 3×3 grid where two players take turns marking X and O. The program checks for a winner or declares a draw after every move.

## 🚀 Features
- Two-player mode
- Win detection (horizontal, vertical, diagonal)
- Draw detection
- Simple and interactive gameplay
